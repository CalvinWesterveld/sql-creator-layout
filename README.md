# Habbo SQL Creator
Lightweight and easy to use SQL Creator with the support of multiple emulators.

##### Currently supports:
- Arcturus

### Requirements
You can install composer [here](getcomposer.org).
- composer

### Installation
```shell
$ composer create-project tafelglotzer/sql-creator-layout --stability dev
```
